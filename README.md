<h3>Command-line:</h3>
<ul>
<li>You can start script by this command: --[group] --script script --token token.
<ul>
<li>For example: --friends -s 1 --token your_token</li>
</ul>
</li>
<li>You can get list of scripts by this command: --[group] --list</li>
</ul>
